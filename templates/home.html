Steps I followed This Far:

01. First created Djando core file with "django-admin startproject core ." the '.' represent the root file, no subfolder will be created.
02. Then we tested the connection by running "python manage.py runserver", which provide us with a link that shows the connection is done.
03. Then we created our first app called /pages by running "python manage.py startapp pages" which created a app called 'pages' in the root
04. Then then we added the app into our core/settings.py -> INSTALLED_APPS.
05. Then we returned 'This is Mehrab And Learning Django' with HttpResponse from django.http inside home function.
06. Then we created a urls.py file for django to know when to use the views.py. We imported path from django.urls and home function from .views.
07. Then we added the url in core/urls.py with include() from django.urls.
08. Then by running the server again with "python manage.py runserver" we printed 'This is Mehrab And Learning Django' 09. Then we created 'templates' folder in the root dir and added that in 'core/settings.py' -> TEMPLATES -> DIR: [BASE_DIR/'templates'].
10. Then we removed our previous http(txt) return from pages/views.py and switched to render(HTML) and returned a dictionary with year.
